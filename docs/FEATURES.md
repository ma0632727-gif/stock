# المميزات - Features

## 1. إدارة شجرة الحسابات ✅
- إنشاء وتعديل وحذف الحسابات
- تصنيف الحسابات (أصول، خصوم، حقوق ملكية، إيرادات، مصاريف)
- دعم الحسابات الهرمية
- رموز حسابية معيارية

## 2. نظام القيود المحاسبية ✅
- قيد اليومية (Journal Entry)
- التحقق من التوازن (Debits = Credits)
- حفظ القيود كمسودات أو نشر مباشرة
- تتبع تاريخ وتوقيت القيود
- الحصول على الإيصالات (Vouchers)

## 3. الدفاتر المساعدة ✅
- الدفتر العام (General Ledger)
- دفتر الأستاذ المساعد (Sub-Ledger)
- دفتر المبيعات (Sales Journal)
- دفتر المشتريات (Purchase Journal)
- دفتر النقدية (Cash Journal)

## 4. القوائم المالية ✅
- الميزانية العمومية (Balance Sheet)
  - تقسيم واضح للأصول والخصوم وحقوق الملكية
  - حسابات فرعية ورئيسية
  
- قائمة الدخل (Income Statement)
  - الإيرادات والمكاسب
  - المصاريف والخسائر
  - صافي الدخل
  
- قائمة التدفقات النقدية (Cash Flow Statement)
  - تدفقات الأنشطة التشغيلية
  - تدفقات الأنشطة الاستثمارية
  - تدفقات الأنشطة التمويلية
  
- ميزان المراجعة (Trial Balance)
  - التحقق من توازن الحسابات

## 5. التحليل المالي ✅
- النسب المالية:
  - نسب السيولة (Current Ratio, Quick Ratio)
  - نسب الربحية (Profit Margin, ROA, ROE)
  - نسب الكفاءة (Asset Turnover, Receivables Turnover)
  - نسب الالتزام (Debt-to-Equity, Interest Coverage)

- تحليل الاتجاهات (Trend Analysis)
  - مقارنة سنوية
  - تحليل الفترات المختلفة
  
## 6. التنبؤ والموازنات ✅
- التنبؤ المالي (Financial Forecasting)
  - الانحدار الخطي
  - المتوسطات المتحركة
  
- الموازنات (Budgeting)
  - موازنة الإيرادات
  - موازنة المصاريف
  - المقارنة مع الفعلي

## 7. التقارير المتقدمة ✅
- تقارير مفصلة بصيغة PDF
- تقارير بأوقات مختلفة (يومي، أسبوعي، شهري، ربع سنوي، سنوي)
- تقارير مقارنة
- تقارير التحليل العميق

## 8. الأمان والتحكم ✅
- نظام المستخدمين والأدوار
- صلاحيات الوصول (Permissions)
- سجل التدقيق (Audit Trail)
- حماية البيانات

## 9. التقارير الضريبية ✅
- حساب الضرائب المستحقة
- تقارير الضرائب الشهرية
- تقارير الضرائب السنوية

## 10. دعم العملات والشركات ✅
- دعم العملات المتعددة
- نظام متعدد الشركات
- تحويل العملات
- تقارير موحدة
