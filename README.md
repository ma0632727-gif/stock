# نظام الحسابات المتكامل
## Accounting System - Complete Financial Management

نظام حسابات متكامل يشمل شجرة الحسابات، القيود المحاسبية، القوائم المالية، والتحليل المالي والتنبؤ.

### المميزات الرئيسية:
- ✅ شجرة الحسابات المعيارية
- ✅ نظام القيود والقيد اليومي
- ✅ الدفاتر المساعدة
- ✅ القوائم المالية (الميزانية، حساب النتائج، التدفقات النقدية)
- ✅ التحليل المالي (النسب المالية)
- ✅ التنبؤ والموازنات

### التقنيات المستخدمة:
- **Backend:** Node.js + Express
- **Frontend:** React
- **Database:** PostgreSQL
- **ORM:** Sequelize
